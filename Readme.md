Built using **HTML5 Canvas and JavaScript**, the simulator demonstrates real-time **space animation, particle motion, and portal visualization**.

---

# 🌌 Project Overview

The **Wormhole Portal Simulator** recreates the visual effect of traveling through a wormhole in space.

The simulation includes:

⭐ Warp-speed starfield motion
🌀 A glowing vortex portal at the center
💫 Energy rings and spiral portal effects
🌌 A hyperspace tunnel illusion

Clicking anywhere on the screen increases warp intensity by spawning additional stars.

# 🧠 How the Simulation Works

The simulation uses a **simple 3D starfield illusion**.

• Each star has an **x, y, z position**
• Stars move toward the viewer by decreasing their **z value**
• Perspective scaling creates the warp-speed effect
• A radial gradient renders the **wormhole portal**

Together these elements create a **hyperspace travel illusion**.
